== Description ==
**RuleForge for Woo Lite** helps store owners add common **WooCommerce fees** and **discount rules** without code. Point-and-click rules apply at cart/checkout.

**Examples**
- **Surcharge by payment gateway** (e.g., COD 2% cap $10).  
- **Discount for B2B user role** on weekdays.  
- **Handling fee** for perishable category in summer.  

**Features (Lite)**
- Up to **3 active rules**.
- Conditions: subtotal, product in cart, user role, country, gateway, day-of-week, first order.
- Actions: fee fixed, fee %, discount %.
- Simple rule editor with sample buttons.

**Pro adds**
- Unlimited rules with priority/stacking controls.
- Import/Export JSON & one-click recipe presets.
- Preview with mock cart; analytics (“Revenue influenced by rules”).

== Frequently Asked Questions ==
= Where do I find the rules? =
Go to **WooCommerce → RuleForge Lite** and add/edit “Fee Rules.”

= Can I add a payment gateway fee? =
Yes—use an action of **fee %** and optionally set a cap.

= Will this work with variable products and coupons? =
Yes—conditions run against the cart context before totals.

1. Upload the ZIP via **Plugins → Add New → Upload Plugin**.
2. Activate (requires WooCommerce).
3. Go to **WooCommerce → RuleForge Lite**.
